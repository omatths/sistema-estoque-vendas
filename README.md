# 📦 Sistema de Estoque e Vendas

Este é um sistema desenvolvido em terminal (linha de comando) focado no controle de produtos, gerenciamento de vendas e geração de relatórios analíticos de estoque.

## 🚀 Como Executar o Projeto

1. Certifique-se de ter o **Python 3.x** instalado na sua máquina.
2. Baixe todos os arquivos do repositório e coloque-os na mesma pasta:
   * `main.y`
   * `produto.py`
   * `estoque.py`
   * `arquivos.py`
3. Abra o terminal na pasta do projeto e execute o comando:
   ```bash
   python main.py
